# raden-diazka.github.io

Website portofolio pribadi saya sebagai siswa **Teknik Jaringan Komputer dan Telekomunikasi (TJKT)**.

🔗 **Live:** [raden-diazka.github.io](https://raden-diazka.github.io)

---

## Tentang Proyek Ini

Website ini dibuat untuk mendokumentasikan perjalanan belajar saya di bidang jaringan komputer dan keamanan siber. Dibuat dari nol menggunakan HTML, CSS, dan JavaScript — tanpa framework.

---

## Halaman

| Halaman | Deskripsi |
|---|---|
| **Home** | Intro singkat dengan animasi particle background |
| **About** | Bio, skills, dan timeline perjalanan belajar |
| **Portfolio** | Kumpulan proyek dan dokumentasi lab |
| **Certificates** | Sertifikat dari Cisco, IBM, dan ID-Networkers |
| **Contact** | Form kontak via Formspree |

---

## Stack

- **HTML5** — Struktur dan konten
- **CSS3** — Styling, animasi, dark/light mode
- **Vanilla JavaScript** — Interaktivitas, particle canvas, scroll reveal
- **[Typed.js](https://github.com/mattboldt/typed.js/)** — Efek teks mengetik di hero section
- **[Formspree](https://formspree.io/)** — Backend form kontak tanpa server

---

## Fitur

- Dark / Light mode dengan preferensi tersimpan di localStorage
- Animated particle background (canvas) di hero section
- Scroll reveal animation pada semua section
- Card mouse glow effect
- Responsive — mobile-friendly
- Skill cards dengan badge level (Beginner / Intermediate / Advanced)
- Certificate card dengan tombol copy nomor sertifikat
